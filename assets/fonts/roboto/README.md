Bu klasor yerel Roboto TTF font dosyalari icindir.

Gerekli dosyalar:

- Roboto-Regular.ttf
- Roboto-Medium.ttf
- Roboto-Bold.ttf
- Roboto-Black.ttf

CSS baglantisi `assets/css/site.css` icinde `@font-face` ile yapilir.

CDN kullanilmaz. Font dosyalari yoksa tarayici sistemde kurulu Roboto, Segoe UI veya sans-serif fonta duser.
