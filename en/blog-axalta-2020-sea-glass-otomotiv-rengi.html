Axalta 2020 Global Automotive Color: Sea Glass

A practical Set Otomotiv note on Sea Glass, automotive color trends and paint application sensitivity.

Industry News2026-04-224 min read
AxaltaSea GlassAuto Paint

Color summary

Axalta highlighted Sea Glass for 2020. Its color character is: green-blue with calm premium character.

Sea Glass combines green-blue character with calmness, confidence and a customized premium appearance.

What it means for Set Otomotiv

Trend colors like Sea Glass are useful references when explaining color character and guiding special shade requests.

These colors should be evaluated together with lighting, panel angle and existing paint aging before local repair, part painting or full repaint decisions.

Application note

Green-blue tones can suggest sea, glass and technology, so expectations should be clarified with the customer.

On effect shades, neighboring panel differences should be detected early.
